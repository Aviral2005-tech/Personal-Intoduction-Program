# Personal Introduction Program

## Description
This Python program asks the user for their name, age, and hobby, then displays a friendly welcome message.

## Features
- Uses `input()` to get user information
- Stores user responses in variables
- Uses `print()` to display output
- Asks at least 3 questions
- Provides a friendly and welcoming message

## How to Run

1. Make sure Python is installed on your system.
2. Open a terminal in the project folder.
3. Run the program:

```bash
python personal_intro.py
```

## Example Output

```text
What is your name? Aviral
What is your age? 20
What is your hobby? Coding

Welcome, Aviral!
Hello, Aviral!.
You are 20 years old.
Your hobby is Coding.
It's nice to meet you, Aviral!
```
